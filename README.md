# Lumi App

New control SPA for Lumifera

http://lumifera.local/lumiapp/

## Useful commands

### Add Tailwind/shadcn ui component
```shell
pnpm dlx @shadcn/ui@latest add
```

### Build
```shell
pnpm build
```

### Local Dev
```shell
pnpm dev
```
